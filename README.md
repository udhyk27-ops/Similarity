# Contest Submission Verification System

## Overview
공모전 운영 과정에서 응모작의 중복 제출 여부를 확인하고 관리하는 일은  
공정성과 신뢰를 유지하기 위해 매우 중요한 요소입니다.  

이 프로젝트는 공모전 응모작을 체계적으로 관리하고,  
이미 제출된 작품과의 유사도를 기준으로 중복 여부를 검증할 수 있는 시스템을 목표로 개발되었습니다.  
운영자가 많은 응모작을 효율적으로 확인할 수 있도록, 관리 흐름의 단순화에 중점을 두었습니다.

---

## Key Features
- 공모전 응모작 등록 및 관리
- 기존 응모작과의 유사도 기반 중복 검증
- 검증 결과를 한눈에 확인할 수 있는 관리 화면
- 운영자 중심의 직관적인 관리 흐름

---

## Focus
이 시스템은 단순한 검증 기능 구현보다  
운영자가 실제로 사용하는 상황을 기준으로 흐름을 설계하는 데 초점을 두었습니다.  

응모작 수가 많아질수록 발생할 수 있는 관리 부담을 줄이기 위해  
데이터 구조와 화면 흐름을 정리하고,  
중복 검증 결과가 명확하게 전달되도록 구성했습니다.

---

## What I Learned
- 대량의 데이터를 효율적으로 관리하는 구조 설계
- 유사도 기반 검증 기능을 운영 관점에서 활용하는 방법
- 관리 시스템에서 명확한 정보 전달의 중요성
